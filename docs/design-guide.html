Question 12
What is the time complexity of binary search in a sorted array?
A O(n)
B O(log n) ✓
C O(n log n)
D O(1)
Explanation
Binary search works by repeatedly dividing the search interval in half. It compares the target value to the middle element of the array. Since the search space is halved at each step, the time complexity is O(log n).
Question Card — Incorrect Answer
Question 13
Which data structure uses FIFO (First In, First Out) ordering?
A Stack ✗
B Queue ✓
C Binary Tree
D Hash Map
Explanation
A Queue follows FIFO ordering — the first element added is the first one removed. A Stack, on the other hand, follows LIFO (Last In, First Out) ordering.
